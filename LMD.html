What is DML?

DML (Data Manipulation Language) is used to manage data inside database tables.

It includes INSERT, UPDATE, DELETE, SELECT operations.

🏗️ DDL = structure | 🧑‍💻 DML = data

INSERT

INSERT adds new data into a table.


INSERT INTO students (id, name)
VALUES (1, 'Ali');

UPDATE

UPDATE modifies existing data in a table.

            
UPDATE students
SET name = 'Ahmed'
WHERE id = 1;

⚠️ Always use WHERE

DELETE

DELETE removes data from a table.


DELETE FROM students
WHERE id = 1;

⚠️ Without WHERE → deletes everything

SELECT

SELECT retrieves data from database.


SELECT * FROM students;

WHERE

WHERE filters data using conditions.


SELECT * FROM students WHERE age > 18;

ORDER BY

ORDER BY sorts results.


SELECT * FROM students ORDER BY age DESC;

GROUP BY

GROUP BY groups same values.


SELECT age, COUNT(*) FROM students GROUP BY age;

INNER JOIN

INNER JOIN connects multiple tables.


SELECT s.name, c.course
FROM students s
INNER JOIN courses c
ON s.id = c.student_id;
